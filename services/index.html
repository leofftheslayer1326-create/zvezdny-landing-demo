Прогулки

Звёздный пикник.
Скандинавская ходьба.

По запросу — корзина для пикника на Медовые водопады или верхние терренкуры Курортного парка. Скандинавская ходьба — программа для тех, кто приехал не только лежать на шезлонге.

  • Корзина с собой — собирает шеф
  • Палки для ходьбы — выдаём на стойке
  • Маршрут согласовываем с врачом санатория
  • Групповые выходы — по утрам
